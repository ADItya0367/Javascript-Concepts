By defining methods in objects (constructor method) we can modify the details see as below 

Creating the details of employees 

<script>
function emp(id,name,salary)
{
 this.id=id;
 this.name=name;
 this.salary=salary;
}
e={100,"aditya",20000};
document.write(e.id+" "+e.name+" "+e.salary);
</script>

THIS IS THE SIMPLE EXAMPLE OF CREATING OBJET THROUGH CONSTRUCTOR
